# yals
Yet Another LS

Me learning Rust via doing a ls rewrite.

## Usage
```
Usage: yals [OPTIONS]

Options:
  -a, --all             Print all files
  -R, --recursive       Print recursively all directories
  -l, --long            Use a list format
  -s, --size
  -H, --human-readable  Pretty print file sizes
  -1                    list one file per line
  -t                    sort by time
  -h, --help            Print help
```
